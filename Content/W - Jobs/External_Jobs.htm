Creating external jobs

External jobs are those you create manually, typically for jobs you've received through methods other than the Leads.

Note:

New manual jobs can only be created for customers already in your customer list. You cannot create new customers through the Create a new job option.

To add a new customer, use the hipages tradiecore app, where you can manage both new and existing customers. The full job creation experience will be available soon on hipages tradiecore web!

To add a job manually to hipages tradiecore, follow the steps given below:

  1. Select Create + on the Jobs list.

    The Create a new job pop-up will appear.

  2. In the Customer name field, enter the customer’s name.

    The other fields will auto-populate with the customer’s details.

  3. Once you select a customer, a new job is automatically created, and you’ll be redirected to the new job page.

  4. From here, you can add further details, such as the job location and description, and start sending quotes and invoices to the customer.
